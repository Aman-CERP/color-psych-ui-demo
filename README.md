# Aether Lab • Color Psychology UI Demo

**Interactive React demo for corporate color psychology testing**

A premium studio-quality demo exploring calm vs warm color accents, dark/light modes, glassmorphism, and live user feedback on emotional impact.

## Live Demo Features
- 4 switchable corporate palettes (Aether Calm, Ember Professional, Verdant Studio, Lumina Calm)
- Full dark/light mode with persistence
- Glassmorphism + smooth micro-interactions
- Interactive Color Psychology Lab
- Real-time feedback & averages
- Research-backed design (2025-2026 UX studies)

Clone and run locally to test how different color palettes affect perceived calmness and premium feel.

```bash
npm install
npm run dev
```
